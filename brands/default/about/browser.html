Browser Compatibility and Support

At time of writing (15/11/17) Journal Archives is supported for use on the following browsers:

On Windows 7:

  • Microsoft Internet Explorer 11
  • Mozilla Firefox 56
  • Google Chrome 62

On Mac OSX Sierra

  • Safari 10.1

The following browsers are unsupported but working at time of writing:

  • Microsoft Edge
  • Microsoft Internet Explorer 10

The following browsers are unsupported and do not work on the Journal Archives service:

  • Microsoft Internet Explorer 9
  • Microsoft Internet Explorer 8
  • Safari 9.1.3 on Mac OSX

We only consider supporting Microsoft's Windows (desktop) supported browsers for security reasons.
More information on which browsers are included can be found at the following page on the Microsoft website: https://support.microsoft.com/en-gb/help/17454/lifecycle-faq-internet-explorer

All compatibility testing was undertaken using Browserstack.

Mobile browsing

The Journal Archives service is not presently optimised for mobile devices.
A mobile-friendly build of the service is currently in development.
