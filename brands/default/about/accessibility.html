Accessibility information for Journal Archives

Q. What is the maximum font size and does text reflow when you enlarge the font?

A. There is no maximum font size, and yes - text does reflow in the new responsive template.


Q. Can the background/foreground colours or contrasts be changed, and if so, how?

A. If the researcher uses a high contrast browser plugin like the one found here, then the site uses standard CSS so views will change to be high contrast.


Q. Are there keyboard-only equivalents for all mouse actions? Where could I find a list? Are there short cut keys to reduce tabbing round links?

A. No, only standard controls are available.


Q. Is text marked up so it can be navigated in a meaningful way (for example by heading level)?

A. Text markup is standard and semantic where possible.


Q. Can text be selected and read by text-to-speech tools?

A. Yes, with the proviso that Journal Archives is a Javascript application and the text-to-speech tools must understand DOM structure (so the text-to-speech tools must be recent).


Q. Are text descriptions available for all relevant graphics and images?

A. All graphics and images on our Home, About and Support pages now have text descriptions.


We hope this information is useful. For any enquiries, please contact the helpdesk at journalarchives@jisc.ac.uk
