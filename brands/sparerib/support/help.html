This is the Help page for the Spare Rib section of Journal Archives. For the Help page for the main Journal Archives service, click here.

This page is intended to assist researchers using Spare Rib specifically on the Journal Archives platform. It is possible to search all of Journal Archives (including Spare Rib) via the normal landing page here, but also to search on Spare Rib only via it's own landing page, here. This Help page contains information about the Spare Rib-specific parts of the service.

Navigation bar

There is a black horizontal navigation bar at the top of the screen on every page of Journal Archives, so you can quickly navigate to any part of the website from any page you are on. Some options are only available on certain pages - for example the Download and Share buttons are only enabled when you have opened an article or issue.
If you are using the Spare Rib landing page, the Home, About and Support links from the Navigation bar will direct you to Spare Rib-specific versions of those pages.
If you wish to access the main Journal Archives site, you can click the link marked “Search the whole of Journal Archives” from the Spare Rib landing page, or click here.

Landing Page

The complete run of the Feminist journal “Spare Rib” was added to the Journal Archives service in May 2015. Spare Rib is open to all - no subscription is required to access the Spare Rib content.
Spare Rib has it's own separate homepage within Journal Archives found at https://journalarchives.jisc.ac.uk/britishlibrary/sparerib to accomodate extra functionality for this collection.

From the Spare Rib homepage, searches only explore the Spare Rib archive (as directed on the page). You can scroll down the page to see the issue covers, and click them to open a whole issue in the Viewer.
Please note that whole issues do not have the full text available in the Details panel when viewed this way, although the full text is present for their individual articles. Whole issues can also be downloaded as a single PDF file if desired.
Whilst the full text is not available in the Details panel for full issues, the PDF files for both the full issues and articles can be searched using an external PDF reader such as Adobe Reader if required.

Logging in to Journal Archives

To accommodate the addition of the Spare Rib journal in 2015, the Search feature of the Journal Archives service has been made open to all.
As such, it's possible for anybody to search the complete catalogue, although only the Spare Rib content is open-access for viewing and downloading.
Without authentication, it isn't possible to open results in the Viewer or download them as PDFs if they aren't part of the Spare Rib archive.
Readers searching on the Spare Rib section of Journal Archives will not encounter any results that require authentication in order to be accessed.
To access the rest of the content on the main Journal Archives service, a valid authentication is required. You can read more about this here.

Searching

Standard search

We are using Elasticsearch open source software to index the data in Journal Archives.

To run a standard search enter your search terms in the search box and select the "search" button. By default, this will search all of the information in the bibliographic records as well as the full text. The number of matching results will be shown to the right of the search box.

If you do want to limit your search to a particular field, for example Article Title or Author, you can do so using the drop-down list to the left of the search box.

The options available are:

  • All
  • Article Title
  • Author

Advanced Search

Simply select "more search options" on the search page to show additional search features.

From here you can specify date range, volume, issue, and start page. You can also add new rows and specify whether your terms MUST, SHOULD or MUST NOT appear in your results.

The first clause of the search (in the main search box) is a MUST - so the results must include whatever goes in that first box. This means that whatever else goes in the search you are essentially starting from a situation where the search in the first box is compulsory. This means that adding a SHOULD clause to this cannot increase the set of results. Adding a SHOULD clause cannot reduce the set of results either (as the clause is optional).

This means that adding a SHOULD clause does not change the total number of results. What it does do is re-order the results when using the relevance ranking, so those items matching the SHOULD clauses come to the top.

Operators, regular expressions and reserved characters

Boolean operators

The familiar boolean operators AND, OR, NOT (alternatively you can write them as &&, ||, !) are supported and can be entered directly into the search box. NOT takes precedence over AND, which takes precendence over OR and you can combine these operators with brackets. These operators can affect the terms to the left and right of them.

You can also use the prefered Elasticsearch operators ‘+‘ (which means the term MUST be present) and ‘-’ (which means the term MUST NOT be present). These operators can only affect the terms to the right of them.

The example below finds all results for "shakespeare" without "venice" in.

                shakespeare -venice

By default, any terms you enter in the search box MUST be present. For instance, all of the searches below are identical:

           	shakespeare -venice ode
            	+shakespeare -venice ode
            	+shakespeare -venice +ode

Wildcard operators

Two wildcard operators are available:

                * = Match any character, including no character
                ? = Match any single character; will not match no character

Please avoid wildcard characters occurring at or before the third position as they may produce too many unrelated matches.

Reserved characters

Certain characters function as operators in the system and if you want to use them in your search query, but not as an operator, then you will need to escape them with a leading backslash.

The reserved characters are:

                + - && || ! ( ) { } [ ] ^ " ~ * ? : \ /

Results, Filtering and Browsing

The results of a search can be browsed by clicking the tabs along the top of the Search results list. Results can be browsed by Articles (the default setting) or Year of Publication.




Results can be filtered by Author, Type, Section, or Year of Publication using the drop-down options to the left side of the screen. The twenty options with the most results will be shown under each filter. Click the “remove” link to clear a filter. Multiple filters can be combined if desired.


The search results view can be toggled between a tile- or list- type of view. Thumbnail images for results can be toggled on or off, and results can be reordered by Relevance, Year of Publication (Ascending or Descending), Author (A-Z) or Title (A-Z)



It is also possible to browse the entire Spare Rib collection by Year of Publication, without any search terms being entered, by clicking the drop-down “Browse” icon from the toolbar.
Please note that when using either the results or catalogue browse features from the Spare Rib homepage, there will be no other journals present in the Journal Browse.


Viewer

To view an article simply select a record from your results list or the browse pages by clicking on the thumbnail or title hyperlink for it. This will open up the Viewer, enabling you to view the article, full text and thumbnails side by side. Using the Navigation bar at the top of the screen, you can download the article as a PDF, as well as download the bibliographic citation information (RIS) and get a permanent URL for the publication when viewing an image.

The Viewer is comprised of three panels: a central panel displaying the images; a Search panel to the left; and a Details panel to the right, which are collapsed by default. To open the left hand panel select the "Search" button on the left-hand side of the viewer and to open the right-hand panel select the "Details" button on the right of the viewer.

You can have all three panels open at the same time, and you can re-size the Search and Details panels by dragging the handle in the middle of the panel edge to the left or right. To close either the Search or Details panel click on the cross at the top of the panel edges.

The options available on each of the three panels are listed below:

Image panel

The central panel in the Viewer displays the page images. There is a toolbar on the left hand side of the panel providing the following options:

  • Zoom in
  • Reset zoom (reset the zoom level so the image is the original default size)
  • Zoom out

Search Panel

To open the Search panel, select the "Search" button on the left had side of the viewer. You can re-size the panel by dragging the handle in the middle of the panel edge across to the right and you can close the panel using the cross at the top of the panel edge. There are three tabs providing options for navigating publications:

  • Pages (this lists pages matching your search terms so you can select one to jump straight to that page)
  • Thumbnails (this displays thumbnail images for every page in the article.)
  • Publications (this displays all of the other articles in the platform that match your search terms, so you can quickly jump to a different article from within the Viewer instead of returning to the results page to select another article.)

Details panel

To open the details panel click on the "details" button on the right hand side of the Viewer. You can re-size the panel by dragging the handle in the middle of the panel edge across to the left and you can close the panel using the cross at the top of the panel edge. There are two tabs providing additional information on the publication:

  • Details (this displays the bibliographic information for a publication)
  • Full text (this displays the full text for the article).

Full text is available for all articles.

Downloading

You can download an article using the "Download" link underneath each search result or by using the download icon on the toolbar in the Viewer while viewing an article. You can also download the bibliographic information in .RIS format using the download icon on the toolbar.

Please note: the Download button is only enabled on the Navigation bar when you have opened an article in the Viewer. On all other pages, it is currently disabled and greyed out.

Stable URLs

The "Share" button on the Navigation bar at the top of the screen provides a stable (permanent) URL for the article you are viewing for you to copy and paste into your documents or web pages. The button is deactivated and greyed out until you open an article in the Viewer, at which point it is enabled and when you select it a dialogue box containing a stable URL for the publication will be displayed.

When you select the button the dialogue box below will open, displaying a stable URL for the publication for you to copy and paste into a web page or document.
